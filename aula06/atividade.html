1. O que é DOM?

O DOM (Document Object Model) é uma representação da página HTML em forma de estrutura de árvore. Quando o navegador carrega um HTML, ele transforma os elementos (tags) em objetos que o JavaScript pode acessar e modificar.

2. O que é um nó (node)?

Um nó (node) é qualquer item dentro do DOM. Tudo na página é considerado um nó.

3. Métodos de seleção

  • Seleciona um elemento pelo ID.
  • document.getElementById("titulo");
  • querySelector()
  • Seleciona o primeiro elemento que corresponde a um seletor CSS.
  • document.querySelector(".classe");
  • querySelectorAll()
  • Seleciona todos os elementos que correspondem ao seletor.
  • document.querySelectorAll("p");

4. Diferença entre innerHTML e innerText

  • innerHTML

    • Permite acessar ou alterar o conteúdo HTML (incluindo tags). element.innerHTML = "Texto";

  • innerText

    • Acessa ou altera apenas o texto visível (ignora tags HTML). element.innerText = "Texto";

5. O que são eventos?

Eventos são ações que acontecem na página e que o JavaScript pode "escutar" e responder.

  • Exemplos:
  • onclick → quando clica
  • onchange → quando um valor muda
  • onmouseover → quando passa o mouse
